PathOs란?

PathOs는 Pathologist Optimal Segmentation의 약자로,
병리학자가 WSI(Whole Slide Images)에서 원하는 세포 및 조직을 빠르게 학습·추론할 수 있도록 돕는
Human-in-the-loop 기반의 노코드 병리 AI 서비스입니다.

노코드 솔루션

직관적인 UI를 통해 프로젝트 생성부터 어노테이션, 모델 학습까지 모든 과정을 코드 없이 수행할 수 있습니다.

최적화된 구조

대용량 병리 이미지에 최적화된 프론트엔드–백엔드 통신 구조를 갖추고 있습니다.

Dual-Branch 구조

Cell과 Tissue를 모두 처리할 수 있는 멀티 파이프라인 구조로, 최소한의 초기 어노테이션만으로도 높은 정확도의 커스텀 모델 학습이 가능합니다.

실시간 세포 분석

AI 기반 병리 분석의
새로운 패러다임

PathOs는 최신 AI 기술을 활용하여 병리학자의 워크플로우를 혁신적으로 개선합니다.
복잡한 코딩 지식 없이도 고성능 AI 모델을 학습시키고 활용할 수 있습니다.

고해상도 의료 이미지 뷰어

십만 픽셀 이상의 병리 이미지를 부드럽게 탐색하며 세포 단위까지 정밀하게 관찰합니다.

사용자 맞춤 AI 라벨링

연구자 정의 라벨로 나만의 AI를 학습시키고, 실시간 분석 결과를 적용합니다.

피드백 기반 모델 고도화

사용자의 판단과 수정이 모델 개선으로 이어지며 (Human-in-the-Loop), 스스로 발전하는 AI를 완성합니다.

데이터 흐름 최적화

PathOs는 대용량 병리 이미지 데이터를 효율적으로 처리하기 위한 최적화된 데이터 흐름 구조를 제공합니다.

이미지 캐싱

자주 사용하는 타일과 레이어를 캐싱하여 반복 접근 시에도 지연 없이 빠른 렌더링을 제공합니다.

분산 처리

여러 서버에 분산된 연산 처리를 통해 대용량 이미지도 실시간 분석 성능을 유지합니다.

계층적 처리

다양한 해상도의 이미지를 계층적으로 처리하여 효율성을 극대화합니다.

정규화된 데이터 구조

이미지, 메타데이터, 어노테이션을 일관된 구조로 관리하여 확장성과 유지보수성을 높입니다.
